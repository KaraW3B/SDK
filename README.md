# KaraW3B SDK 

The KaraW3B SDK aims to simplify integration of KaraW3B in thirdparty clients

## 📦 Packages

### KaraW3B.SDK
Contains basical stuff like models and some helpers

### KaraW3B.SDK.Client
Contains connectors to easily call KaraW3B server API
